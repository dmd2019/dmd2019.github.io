如何在激烈竞争中寻找产品定位（一）行业分析

发表于 2019-06-03

该系列文章是对王诗沐《幕后产品》一书的学习记录与理解总结。读此书的目的很明确，一是希望学习借鉴诗沐所总结的做产品的整体思路，以形成自己的基本思考框架；二是希望吸收诗沐这一思路下对网易云音乐操刀的实际案例讲解，来作为这个思考框架下的基本内容填充。
这篇文章就是该系列的第一篇，关于产品定位中的行业分析。

做产品定位，有一套基本的方法如下，有三个步骤。

  1. 分析行业、市场、竞争对手等外部环境，从抽象到具象一步步剖析产品现在所处的环境
  2. 结合外部的分析和自身用户群、优势的分析，找到产品打开市场的切入口。
  3. 在切入点的基础上，对产品定位和长期发展做出阶段规划，并设计扩张的接口。
  4. 其中整个方法最关键的在于“变化”二字，机会往往出现在变化之中。

机会往往出现在未来发生巨大变化的行业中，所以思考产品定位的第一步，就是搞清楚自己产品所在行业的情况！这是发现机会的开始。

那么如何分析一个行业的情况，可以拆解为以下三个问题。

  1. 这个行业有哪些玩家？他们之间的关系是怎样的？以及行业现状？
  2. 未来几年，这个行业会发生变化吗？会发生什么变化？这些变化中会产生什么机会？
  3. 在行业里的玩家会如何抓住变化中的机会？

下面时间回到2013年，以音乐行业为例进行分析。

这个行业有哪些玩家？他们之间的关系是怎样的？以及行业现状？

玩家：

词曲作者等幕后工作者
艺人
经济公司、唱片公司、版权公司
音乐演出公司
唱片店等实体销售渠道
电台DJ、乐评人、音乐杂志等媒体
在线音乐平台
乐迷、听众
游戏、广告、影视、彩铃等相关获利者

他们之间的关系是怎样的？

首先音乐是一种内容。内容是这个行业的源头，因此这个行业最关键的就是内容。

因此，处于音乐行业上游的几类玩家也是围绕音乐内容来明确自己的位置的。作词作曲编曲等幕后人员是音乐作品的灵魂；艺人给音乐作品打上个人标签；经济人、唱片公司、版权公司则帮助艺人制作音乐、发行音乐、推广音乐、传播音乐、明确音乐版权，以及做音乐的营收。

整体来说，音乐行业可以分为三类玩家。

内容源部分： 词语作者幕后人员、艺人
中间部分： 涉及音乐的发行、传播、产生营收等诸多环节，从经纪人、唱片公司到演出公司、在线音乐平台都归为此部分
听众部分： 听众

其中，中间部分是帮助上游的内容源部分更好地输送内容到下游的听众部分，并在整个过程中获取利润。在80、90年代，唱片店、音乐电台/排行榜、大产品公司都把持过中间部分，在音乐行业中发挥了举足轻重的影响力

行业现状？

过去，在CD销售的黄金年代，整体行业利润非常丰厚。

现在，自从互联网开始兴起，数字音乐逐渐取代CD后，打破了原本稳固的商业模式，由于音乐版权管理不善，新的商业模式也没有建立起来，音乐行业整体收入下降。

未来几年，这个行业会发生变化吗？会发生什么变化？这些变化中会产生什么机会?

（这里需注意变化和机会的关系，变化是客观存在的，机会需主动从变化中去识别，去挖掘。）

会发生变化吗？

首先，对比世界上其他国家。同样受到互联网的冲击，韩国运用行政手段整治盗版，止住了颓势。美国有立法明确下载盗版违法。

其次，类比相近行业。和音乐行业相近的视频行业，同样在互联网冲击下盗版泛滥，但经过一轮正版化洗牌，行业已经处于蓬勃的发展期。

所以基本判断是会发生，并且朝着好的方向变化

发生什么变化？

关键变化是，移动互联网和正版化。

移动互联网的特点是：资源极多、时间碎片化、传播极广。其所带来的变化，概括来说，是移动互联网影响了听众关于使用音乐的方式，进而会引发行业中间部分的变化（即行业的革命）

具体变化的点如下：

  • 听众能随时随地听音乐，且能听全世界的音乐
  • 听众更愿意互动和分享了，微博和微信教育了用户随时随地分享自己的生活状态
  • 互联网让音乐的传播速度变得飞快
  • 用户的影响力比CD时代大了很多。音乐出来后用户马上能评价自己是否喜欢这个音乐
  • 相比以往的电台、排行榜、唱片公司，互联网平台有机会作为中心化平台连接巨量音乐用户

这其中变化的本质是互联网连接了人与人，人与信息后。1.让音乐作为一种信息来说，它的传播方式改变后，传播的速度就变得更快，范围更广了。2.同时连接巨量人与人，人与信息的中心化平台，也让用户有了互动的机会。3.音乐传播方式的改变，音乐行业的利润分配机制也就可能随之改变。

这些具体的变化可以总结为三个方面。音乐传播，音乐互动，利润分配

这个变化中会产生什么机会？

移动互联网。概括来说，一，网络把人与信息连接起来，改变了音乐传播互动的渠道，有减少中间环节的机会。二，传播方式改变，音乐行业会产生更利于内容源的利润分配机制。

行业里的玩家会如何抓住变化中的机会？

内容源部分。
客观的变化是，获得更多互联网的流量，且流量结构直接、扁平；可以直接获得听众的反馈。这些变化同时也带了的机会是能获得直接的听众反馈，便于更快作出反应；流量更加直接、扁平，有自主营销，自主传播的机会；内容长尾化

中间部分。
客观变化是流量渠道结构改变。机会是让听众与内容源更加靠近，取代传统电台、媒体等渠道，成为新的平台，制定新的商业模式。

听众部分。
客观变化是可以听到全世界的音乐；可以更直接的与偶像互动；对自己喜欢的音乐更有话语权。机会是对于音乐，有更大选择自主权。
